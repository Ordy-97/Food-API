API Documentation

This documentation provides an overview of all available API routes grouped by module.

Admin Routes

  • POST /admin/vandor - Create a new vendor
  • GET /admin/vandors - Get all vendors
  • GET /admin/vandor/:id - Get a vendor by ID
  • PUT /admin/delivery/verify - Verify a delivery user
  • GET /admin/delivery/ - Get all delivery users
  • GET /admin/transactions - Get all transactions
  • GET /admin/transaction/:id - Get a transaction by ID

Customer Routes

  • POST /customer/signup - Sign up a new customer
  • POST /customer/login - Log in as a customer
  • PATCH /customer/verify - Verify customer account
  • GET /customer/otp - Request OTP
  • GET /customer/profile - Get customer profile
  • PATCH /customer/profile - Edit customer profile
  • POST /customer/cart - Add items to cart
  • GET /customer/cart - Get cart items
  • DELETE /customer/cart - Delete cart items
  • GET /customer/offer/verify/:id - Verify an offer
  • POST /customer/create-payment - Create a payment
  • POST /customer/create-order - Create an order
  • GET /customer/orders - Get all orders
  • GET /customer/order/:id - Get an order by ID

Delivery Routes

  • POST /delivery/signup - Sign up a delivery user
  • POST /delivery/login - Log in as a delivery user
  • PUT /delivery/change-status - Change service status
  • GET /delivery/profile - Get delivery user profile
  • PATCH /delivery/profile - Edit delivery user profile

Shopping Routes

  • GET /:pincode - Get food availability by pincode
  • GET /top-restaurants/:pincode - Get top restaurants by pincode
  • GET /foods-in-30-min/:pincode - Get foods available in 30 minutes
  • GET /search/:pincode - Search for foods by pincode
  • GET /offers/:pincode - Get available offers by pincode
  • GET /restaurant/:id - Find a restaurant by ID

Restaurants Routes

  • POST /vendor/login - Vendor login
  • GET /vendor/profile - Get vendor profile
  • PATCH /vendor/profile - Update vendor profile
  • PATCH /vendor/service - Update vendor service status
  • PATCH /vendor/coverimage - Update vendor cover image
  • POST /vendor/food - Add food items
  • GET /vendor/foods - Get all food items
  • GET /vendor/orders - Get all current orders
  • PUT /vendor/order/:id/process - Process an order
  • GET /vendor/order/:id - Get order details by ID
  • GET /vendor/offers - Get vendor offers
  • POST /vendor/offer - Add a new offer
  • PUT /vendor/offer/:id - Edit an offer
